MMM-POSpotify

Modern Spotify Module for MagicMirror²

1

Create a Spotify App

Visit the Spotify Developer Dashboard and create a new app. Use these settings:

App name: MagicMirror POSpotify
App description: Personal MagicMirror Spotify Display
Redirect URI: http://127.0.0.1:8100/callback

Important: Use exactly 127.0.0.1 (not localhost) for security compliance!

2

Configure the Module

Add your Spotify app credentials to your MagicMirror config:

{ module: "MMM-POSpotify", position: "top_right", config: { clientID: "YOUR_CLIENT_ID", clientSecret: "YOUR_CLIENT_SECRET", // Other options... } }
3

Authorize Your Account

Click the button below to connect your Spotify account. You'll be redirected to Spotify to grant permissions.

Permissions requested: Read playback state, control playback, view currently playing track
